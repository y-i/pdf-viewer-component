# pdf-viewr-component
Web Components for PDF

# How to use

## Develop

- `npm run install`
- `npm run dev`
  - `sample-slide.pdf` is required to show a PDF by default

## Build

- `npm run install`
- `npm run build`

## Preview

- `npm run install`
- `npm run preview`
  - `sample-slide.pdf` is required to show a PDF by default
